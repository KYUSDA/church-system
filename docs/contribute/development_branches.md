## Development

To contribute:

1. Pull latest:

git pull origin `v2-staging`


2. Create a feature branch:

git checkout -b feature/<team>/<description>


3. Work on the task  
4. Push and submit PR into `v2-staging`  

5. Request a review from a teammate.

6. Merge after approval.
