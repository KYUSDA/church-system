# KYUSDA Website v2.1 Release Notes

## Introduction

We are thrilled to announce the release of **Website v2.1**, marking the second major version of the KYUSDA platform! This update introduces **exciting new content sections**, **enhanced user engagement features**, and **significant UI improvements** to better serve our community. Building on the solid foundation of v1, v2 focuses on **content expansion, accessibility, and modern design**, creating a more immersive and informative experience for all users.

### Version: v2.1

### Release Date: April 12, 2026

### Deployment Platform: Azure (Dockerized Deployment)

## New Features & Enhancements

This release brings several innovative features and enhancements to the KYUSDA website, expanding our digital presence and community resources:

### **Church Magazine Section**

- Introduced a dedicated **Church Magazine** area for sharing community updates, announcements, and inspirational content.
- Features easy-to-navigate articles and multimedia content to keep members informed and engaged.

### **Resources Section**

- Added a comprehensive **Resources** hub providing helpful materials, guides, and tools for church members.
- Includes downloadable documents, educational content, and quick-reference materials for spiritual growth.

### **Blogs Subdomain Launch**

- Launched a dedicated blogs site at **blogs.kyusda.org** for in-depth articles, insights, and discussions.
- Enables community members to share thoughts, testimonies, and educational content in a focused blogging environment.

### **Enhanced Department Detail Pages**

- Expanded department pages with **more detailed information**, including leadership profiles, activities, and contact details.
- Improved navigation and layout for better accessibility and user experience across all department sections.

### **UI/UX Updates**

- Refreshed the overall user interface with modern design elements, improved responsiveness, and enhanced usability.
- Streamlined navigation and visual hierarchy to make the website more intuitive and visually appealing.

## Deployment & Rollout Plan

- **Deployment Date:** April 12, 2026
- **Deployment Method:** Azure DevOps CI/CD Pipelines
- **Downtime Expected:** Minimal (rolling deployment)
- **Rollback Plan:** Revert to v2.0 in case of unexpected issues.

## Known Issues & Workarounds

- ❗ **Issue:** Some users may experience caching issues with the new UI updates.
  - **Workaround:** Clear browser cache or use incognito mode to view the latest changes.
- ❗ **Issue:** Initial load times for the blogs subdomain may be slightly longer during peak hours.
  - **Workaround:** We are monitoring performance and optimizing as needed.
